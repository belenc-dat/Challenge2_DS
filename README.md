# Challenge2_DS
Análisis de Evasión de Clientes (Churn Analysis)##
Descripción del Proyecto
Este proyecto analiza el comportamiento de los clientes de una empresa de telecomunicaciones para identificar los factores clave que influyen en la cancelación de servicios (Churn).

Principales Hallazgos
Tasa de Evasión: Aproximadamente el 26.5% de los clientes han abandonado el servicio.
Tipo de Contrato: Los contratos mensuales (month-to-month) presentan una tasa de deserción significativamente más alta en comparación con los contratos anuales.
Servicios Adicionales: La falta de servicios como TechSupport y OnlineSecurity está correlacionada con una mayor probabilidad de abandono.
Métodos de Pago: Los clientes que utilizan Electronic check tienden a evadir más que aquellos con métodos automáticos.
Tenencia (Tenure): Existe una correlación negativa fuerte; a menor tiempo en la empresa, mayor es el riesgo de que el cliente se vaya.
Tecnologías Utilizadas
Python (Pandas, NumPy)
Visualización: Matplotlib, Seaborn, Plotly
Extracción: JSON normalization desde API
